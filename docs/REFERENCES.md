## 一、K12-KGraph（北京大学，2026年）★★★★★ 最重要参考

- GitHub: https://github.com/haolpku/K12-Dataset
- HuggingFace数据集: https://huggingface.co/datasets/lhpku20010120/K12-KGraph
- 论文: arxiv:2605.09635
- 核心贡献:
  - 7节点9边Schema设计，从人教版教材直接构建
  - 覆盖48本人教版教材 → 6,579概念 + 1,364技能 + 1,171习题
  - 总计10,685节点 + 23,278条边
  - LLM辅助构建流水线：OCR → 分段 → LLM抽取 → 合并 → 验证
  - DAG验证 + 专家审核（Fleiss κ=0.84 一致性）
  
### 7节点类型
1. Subject（学科）
2. Book（教材）
3. Chapter（章）
4. Section（节）
5. Concept（概念/知识点）
6. Skill（技能/解题能力）
7. Exercise（习题）

### 9种边/关系类型
1. is_a - 分类关系（子概念→父概念）
2. prerequisite_for - 前置依赖（必须先学A才能学B）
3. relates_to - 关联关系（概念间语义关联）
4. tests_concept - 习题考查的知识点
5. tests_skill - 习题考查的技能
6. belongs_to - 概念/技能所属节
7. is_part_of - 节属于章、章属于书
8. common_mistake - 知识点的常见错误
9. extension - 拓展知识（超纲但相关）

### 数据规模
- 数学: 最大学科，覆盖1-12年级全部
- 物理: 8-12年级
- 化学: 9-12年级
- 概念粒度: 每个教材平均约137个概念

### 关键启示（可直接用于Kid Agent）
1. 用LLM从教材PDF半自动构建知识图谱，效率极高
2. DAG验证必须做，防止循环依赖
3. 专家审核不可省，LLM抽取有5-10%错误率
4. 数据集可直接从HuggingFace下载导入

## 八、GraphRAG（微软研究院）

- 论文/项目: Microsoft Research
- 核心思路: 向量检索 + 知识图谱结构查询 混合
- 流程:
  1. 用户提问 → 向量检索Top-K候选
  2. 同时从知识图谱结构查询关联节点
  3. 两路结果融合排序
  4. LLM基于检索结果生成回答
- 关键启示:
  - 纯向量检索不够，需要图结构查询补充
  - 融合排序权重: 向量0.6 + 图结构0.4（经验值）

## 九、检索技术方案对比

### 全文检索（SQLite FTS5）
- 优势: 内置、零部署、中文分词可用jieba
- 劣势: 无法理解语义相似度
- 适合: 精确关键词搜索

### 向量检索（LanceDB嵌入式）
- 优势: 理解语义，"分数除法"能匹配"分数的除法运算"
- 嵌入模型: text2vec-base-chinese（本地，免费）或OpenAI embedding（API）
- 向量维度: 768维（text2vec）或1536维（OpenAI）
- 存储估算: 16,000概念 × 768维 × 4字节 ≈ 49MB
- 适合: 学生自然语言提问匹配知识点

### 图结构查询（SQLite递归CTE）
- 优势: 可查前置依赖链、学习路径、关联概念
- SQLite实现: WITH RECURSIVE 递归CTE
- 性能: 百万级边以内无压力
- 适合: "学勾股定理前要学什么""这个知识点关联哪些"

### 推荐组合
```
混合检索 = 向量检索(0.6权重) + FTS5全文(0.4权重)
          → 结果融合 → 图结构扩展（前置+关联）
```

## 十、LanceDB技术细节

- 官网: https://lancedb.github.io/lancedb/
- 特点:
  - 嵌入式向量数据库（类似SQLite之于关系型）
  - 无需独立服务进程
  - 底层Apache Lance列式格式
  - 支持元数据过滤 + 向量检索混合查询
  - Python原生支持
- 安装: pip install lancedb
- 备选方案: ChromaDB（更简单但性能弱）

## 十一、K12-KGraph数据集可直接利用

HuggingFace数据集内容：
- 概念文件: 每个概念的名称、定义、所属章节、重要性
- 技能文件: 解题技能描述
- 习题文件: 题目文本、答案、考查知识点
- 关系文件: 9种边关系
- 覆盖: 数学(全年级)、物理(8-12)、化学(9-12)

导入方式：
```python
from datasets import load_dataset
ds = load_dataset("lhpku20010120/K12-KGraph")
# 然后遍历写入SQLite
```

这是最快的方式，不用人工录入，直接获得6500+概念。
