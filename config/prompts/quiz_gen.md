# 出题Prompt模板

## 知识点信息

- **知识点名称**：{name}
- **年级**：{grade}
- **难度**：{difficulty}（1-5级，1最简单，5最难）
- **知识点描述**：{description}

## 出题要求

1. **题型多样**：可以是填空题、选择题、计算题、应用题
2. **生活化**：题目要贴近生活，有趣味性
3. **难度匹配**：严格按照指定难度出题
4. **避免重复**：不要和学生最近做过的题目重复

## 题型多样性指引

### 填空题
- 适合基础概念和简单计算
- 示例："2×3=___"，"___÷4=5"

### 选择题
- 适合概念辨析和易错点
- 提供4个选项，选项要相似但正确答案唯一
- 干扰选项要设计得合理，反映常见错误

### 计算题
- 适合练习计算能力
- 可以是竖式计算、简便计算等
- 注意数字的选择，便于计算

### 应用题
- 适合综合运用知识
- 要贴近生活场景，有实际意义
- 注意文字简洁清晰

### 图形题
- 适合几何类知识点
- 用文字描述图形，让学生想象
- 示例："一个长方形长8厘米，宽5厘米，它的面积是多少？"

### 推理题
- 适合逻辑思维训练
- 需要多步推理
- 有一定的挑战性

### 游戏题
- 增加趣味性
- 可以设计成找规律、猜数字等
- 激发学习兴趣

## 趣味性要求

1. **使用学生名字**：在题目中使用"小明"、"小红"等学生熟悉的名字
2. **贴近生活场景**：
   - 购物：买文具、买零食
   - 运动：跑步、跳绳、打球
   - 游乐：玩游戏、看电影
   - 动物：养宠物、动物园
   - 节日：生日、春节、儿童节
3. **故事化**：可以设计小故事作为题目背景
4. **互动性**：使用"你"、"我们一起"等词语，增强互动感

## 难度控制说明

### 难度1（简单）
- 基础概念和简单计算
- 一两步就能完成
- 数字要小，便于口算

### 难度2（中等）
- 需要多步计算或综合运用
- 数字适中，需要竖式计算
- 有一定的思维要求

### 难度3（困难）
- 需要多个知识点综合
- 思维要求较高
- 计算量较大

### 难度4（很困难）
- 需要深入理解和灵活运用
- 可能有多种解法
- 容易出错

### 难度5（专家级）
- 超纲或拓展内容
- 需要创新思维
- 挑战性很强

## 返回格式

请用JSON格式返回：

```json
{
    "question_type": "free | choice | calculation | application | game",
    "question": "题目内容（选择题要包含选项）",
    "options": ["选项A", "选项B", "选项C", "选项D"],
    "answer": "标准答案",
    "explanation": "答案解释（给老师看的）"
}
```

**注意**：
- question_type如果是choice，必须提供options
- answer要准确简洁
- explanation要说明解题思路
- 题目要符合指定的难度要求
- 尽量让题目有趣味性
