# 少年助手（Kid Agent）

> 专属孩子的AI成长伙伴 — 小学数学教学 MVP

## 快速开始

```bash
cd ~/projects/kid-agent
pip install -r requirements.txt
cp config/.env.example config/.env  # 填入API Key
python src/main.py --mode cli
```
